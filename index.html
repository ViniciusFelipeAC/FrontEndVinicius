Descomplicando a UML

Unified Modeling Language: A planta baixa do software

O que é UML?

A UML (Linguagem de Modelagem Unificada) é uma linguagem padrão utilizada para especificar, visualizar, construir e documentar os artefatos de sistemas de software.

Por que usar UML?

  • Comunicação: Facilita o diálogo entre equipes.
  • Padronização: Todo mundo entende os desenhos.
  • Planejamento: Evita erros antes de programar.

Principais Diagramas

A UML é dividida em dois tipos:

1. Diagramas Estruturais

Mostram como o sistema é montado (estático).

  • Diagrama de Classes: O mais usado. Mostra as caixinhas e relações.
  • Diagrama de Componentes: Mostra a parte física.

2. Diagramas Comportamentais

Mostram como o sistema funciona (dinâmico).

  • Diagrama de Casos de Uso: O que o usuário faz.
  • Diagrama de Sequência: Mostra a ordem das mensagens.
